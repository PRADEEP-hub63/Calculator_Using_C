# 🧮 Calculator in C

A simple yet robust command-line calculator built using the C programming language. This project demonstrates core programming concepts such as modular design, control structures, input validation, and error handling—making it ideal for beginners and internship-level submissions.

## 🚀 Features

* Perform basic arithmetic operations:

  * Addition
  * Subtraction
  * Multiplication
  * Division
* Menu-driven interface using `switch` case
* Continuous operation using loop (user can perform multiple calculations)
* Input validation to handle incorrect entries
* Error handling for division by zero
* Clean and user-friendly console interface

## 🛠️ Technologies Used

* C Programming Language
* Standard Input/Output Library (`stdio.h`)

## 📂 Program Structure

* Uses **modular programming** with separate functions for each operation
* Implements `switch-case` for operation selection
* Uses loops for repeated execution
* Ensures safe execution with proper validations

## ▶️ How to Run

1. Compile the program:

   ```bash
   gcc calculator.c -o calculator
   ```
2. Run the executable:

   ```bash
   ./calculator
   ```

## 🎯 Learning Outcomes

* Understanding of functions and modular design
* Use of conditional statements (`switch-case`)
* Handling user input and validation
* Basic error handling techniques in C
